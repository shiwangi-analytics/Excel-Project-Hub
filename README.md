This project presents an interactive Sales Data Analytics Dashboard built using Microsoft Excel. The dashboard is designed to analyze and visualize sales performance effectively.

## Key Features
- Data cleaning and preprocessing
- Pivot Tables for data analysis
- Interactive dashboard with charts
- Use of slicers for dynamic filtering

## Insights
- Monthly sales and profit trends
- Top-performing products
- Comparative performance analysis

## Tools Used
- Microsoft Excel

This project demonstrates practical skills in data analysis, data visualization, and dashboard creation using Excel.
